Fonctionnement du programme:

Nous vous invitons à faire compiler le projet via "make" puis de lancer l'excutable avec
comme premier argument "--help"

Une fois le "--help" lu et compris, vous pouvez désormais lancer le programme.
Vous avez un peu plus d'une seconde pour constituer les équipes !

Mais ne vous inquiétez pas, nous vous avons mis à disposition un script sh "test.sh",
il vous permettra de lancer rapidement plusieurs proccess.
Comment l'utiliser ?
./test.sh 'PATH' 'nombre de joueur'

Le 'nombre de joueur' correspond au nombre de joueurs que composent chaque des 5 équipes crées.
Le 'PATH' correspond au point d'encrage des IPCS (man ipc).
